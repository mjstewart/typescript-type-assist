Assign expression to untyped local variable.

This intention assigns an expression to a local variable without type information. If Typescript Type Assist settings has 'End statement with semi colon' enabled, the variable assignment will always end in a semi colon.

The variable declaration type (const, let, var) can be changed in the Typescript Type Assist settings menu.

Useful to be used in conjunction with the existing 'Specify type explicitly' intention built into the IDE.
